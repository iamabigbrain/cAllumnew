03.

timeline

my journey through development, creativity, and growth.

2017

started scratch

discovered programming and fundamentals, began making basic scratch projects

I downloaded scratch 2 and began working on small projects such as "football fish game" and short animation stories, by this time i was 8 years old so i wasnt THAT good.

2019

introduced to roblox studio

i came across roblox studio, where i made small games that were never released

when i started using roblox studio, i saw what code actually looked like and what it's like to work in a proper workspace and how difficult games are to make. this would be useful for unity in the future.

2021

moved to unity

i used unity for the first time and started making an FPS game like karlson.

unfortunately, this wasn't "Big Bills" and was a very basic test of how complicated it is to even make a basic character compared to roblox. some assets ended up being reused 2 years later.

2022

began video editing

i started making connections and began using Davinci Resolve to edit videos.

at this time, i was in the minecraft smp community where a lot of videos were being made and edited. i was one of these people and began learning how to edit from this time.

2023

released my first game

i released big bills july 15th 2023 alongside videos

this project was one of my longest projects as i was learning AI, scripting, modelling and how to use unity all at the same time. this began my journey.
